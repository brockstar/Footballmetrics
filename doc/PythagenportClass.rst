Pythagenport class
==================

Implementation of Clay Davenport's Pythagenport formula. The exponent is:

.. math:: 

	1.5 \cdot \log_{10}\left(\frac{\mathrm{PF}+\mathrm{PA}}{\mathrm{N}}\right) + 0.45
